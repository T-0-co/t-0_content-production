## Purpose

Structural templates for T-0 blog content. Defines the base structure all posts share, plus type-specific variations.

## Base Structure (All Posts)

### 1. Opening (First 100 Words)
Earn attention. Choose one pattern:
- **Scenario Recognition** — Open with a situation the reader has experienced
- **Contrarian Challenge** — Statement that contradicts common belief
- **"What I Learned When..."** — Specific experience that taught something

**Never open with:** Definitions, statistics without context, generic claims, self-answered questions.

### 2. Promise/Thesis (Within 150 Words)
Make clear what the reader will get. Be specific.

### 3. Body Structure
- Lead with the takeaway
- Then provide context
- Then show the work
- Then acknowledge limitations

### 4. Evidence Patterns
- Actual numbers over vague claims
- Specific examples over general advice
- Process over just outcome

### 5. Conclusion/CTA
- **Synthesis + Invitation** — Summarize + invite next step
- **Open Question** — Leave reader thinking
- **Direct CTA** — For practical content

### 6. Length Guidelines

| Content Type | Target Length |
|-------------|-------------|
| Technical/How-To | 1,200–2,000 words |
| Opinion/Analysis | 800–1,200 words |
| Case Study | 1,000–1,500 words |
| Resource/Template | 600–1,000 words |
| Announcement | 300–600 words |

## Content Type Skeletons

### Technical / How-To
Opening → Prerequisites → Overview → Step-by-Step → Verification → Troubleshooting → Next Steps

### Opinion / Analysis
Opening → The Common View → The Problem → Your Perspective → Evidence → Nuance → Implication

### Case Study
Opening (before→after) → Client Context → Challenge → Approach → Results → Learnings → CTA

### Resource / Template
Opening → Who This Is For → What's Included → How to Use → The Resource → Customization → CTA

### Announcement
The News → Why It Matters → Key Details → What's Next → CTA

## Ghost CMS Specifics

- **Title:** 60 chars max (SEO)
- **Meta description:** 155 chars max
- **URL slug:** Short, keyword-relevant, no dates
- **Featured image:** 1200×630px required
- **Tags:** 1 format tag (How-To, Case-Study, Opinion, Resource) + 1-3 topic tags (MCP, Agents, Workflow, Strategy)
- **Timing:** Tuesday–Thursday business hours, avoid Friday afternoon/weekends

## Pre-Publish Checklist
- [ ] Opening earns attention
- [ ] Promise/thesis clear within 150 words
- [ ] Headers enable scanning
- [ ] At least one specific number/example/process detail
- [ ] Claims supported or marked as opinion
- [ ] Meta title <60 chars, meta description <155 chars
- [ ] Featured image set, URL slug clean, tags applied
- [ ] Read opening aloud — would you keep reading?
